**BMI Calculator**

Body Mass Index(BMI) is a measurement of body size based on our height and weight.

World Health Organisation (WHO) recommended BMI Chart:

| BMI Value   |  Weight Category    |
|:------------|:-------------------:|
| Below 18.5  | Underweight         |
|  18.5 - 25  | Normal weight       |
|   25-30     | Overweight          |
|  Above 30   | Obesity             |
